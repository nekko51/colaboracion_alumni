\section{Representation and modelization of data}

\subsection{Aminoacids and properties}
Every aminoacid will be coded as a 21-dimensional vector of real numbers between 0 and 1 yuxtaposed to a 9-dimensional vector of real numbers between 1 and 0. To interpret as follows:
\begin{itemize}
    \item The first list will be the 21-dimensional vector, whose $i$-th component corresponds to the frequency of the $i$-th aminoacid from the forementioned aminoacid string.
    \item The second list will be the 9-dimensional vector, whose $i$-th component corresponds to the frequency of the $i$-th property from the forementioned property list.
\end{itemize}

As you can see, these variables are context dependent, since there is no reference for the frequency of aminoacids or properties. Therefore, if the chain is one that represents an actual protein, all frequencies may be 0 but one of them, which should be 1. Same will happen for the property vector. We will call this a \textit{concrete aminoacid}.

\paragraph{Example:} Let's encode the aminoacid cysteine (\code{\textcolor{green!70!black}{C}}). It is the third in the aminoacid list, thus, the first vector should be $v_{\text{elements} } = (0,0,1,0,...,0)$. Cysteine is hydrophobic and small, then: $v_{\text{properties}} = (0,1,0,0,0,1,0,0,0)$, since hydrophobic is in the 2nd position and small in the 6th.

Now changing the context, another usual setting is the so called \textit{Schrödinger's aminoacid}. That will be the average aminoacid, which elements in both vectors will be *separately* the frequencies of every appearing aminoacid.

*Separately* means that the sum of the components of the first list is 1 and the sum of the components of the second list will also be 1.\futnotetext
\footnotetext{This is true for every kind of aminoacid, whether it is concrete or Schrödinger's.}

\subsection{Chains}

A chain is an array of 298 aminoacids and it represents a whole antibody. Just as before, we will have concrete and Schrödinger's chains.

This is a $298 \times (21 + 9)$ matrix of split row conditioned frequencies.
